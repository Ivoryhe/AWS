Perl Module to run Net::Amazon::EC2

sudo apt-get update
sudo aptitude install libdist-zilla-perl
sudo apt-get install libssl-dev
sudo cpan YAML
sudo cpan Net::SSLeay
sudo cpan IO::Socket::SSL
sudo cpan LWP::Protocol::https
sudo apt-get build-dep libxml-parser-perl
sudo cpan XML::Parser
sudo cpan XML::Simple
sudo cpan Net::Amazon::EC2
sudo cpan VM::EC2::REST::autoscaling
sudo cpan VM::EC2 (you might need to force installation)
sudo cpan install --force VM::EC2

for Cloudwatch
